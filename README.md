# Professional Portfolio - Seif Eldin Tarek

This project is a modern portfolio landing page for Seif Eldin Tarek (FullStack .NET Developer). It includes animation effects, responsive layout, and a presentation-ready summary.

## Files
- `index.html` - main page with advanced sections (sticky nav, skills, timeline, testimonials)
- `styles.css` - modern theme + animation and dark mode styles
- `script.js` - scroll-based reveal, counters, progress bars, dark-mode toggle

## Setup
1. Copy your resume PDF to the project folder and rename it to `resume.pdf`.
   - Source path in your request: `C:\Users\seaif\Downloads\Seif_Eldin_Tarek_ATS_Friendly_FullStack_DotNet_CV (1)-1.pdf.pdf`
   - Set `c:\Users\seaif\Desktop\Portofolio\resume.pdf`
2. Open `index.html` in browser.
3. Optionally edit `index.html` to update contact details, project text, and social links.

## Deployment
- Serve with any static server (GitHub Pages, Netlify, Vercel, or local `Live Server` extension).

## Customize
- Change summary or keywords in hero and about sections.
- Replace placeholder project entries with your actual work.
- For advanced parallax or 3D, add a library like GSAP (https://greensock.com/).
